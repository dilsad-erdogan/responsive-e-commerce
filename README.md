## Used Technologies
**Front-end:** React -> tailwindcss, react, react icons

## Run
Clone the project

```bash
  git clone https://github.com/dilsad-erdogan/responsive-e-commerce.git
```

Go to project directory

```bash
  cd frontend
```

Run project

```bash
  npm run dev
```
